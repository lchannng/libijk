# Libijk

## Overview
A header-only foundational c++ 17 library 

## Quick Start
Prerequisites:
- Asio
- *Optional:* Catch2 (for unittests)

To get started:
```
> git clone https://github.com/lchannng/libijk
> cd libijk
> git clone https://github.com/lchannng/deps-repo deps
> mkdir build
> cd build
> cmake ..
> make
